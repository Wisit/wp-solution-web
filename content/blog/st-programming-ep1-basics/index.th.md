## 1. р╕ир╕╢р╕Бр╣Бр╕лр╣Ир╕Зр╕ар╕▓р╕йр╕▓: Ladder Logic (LD) vs. Structured Text (ST)
р╕Чр╕│р╣Др╕бр╣Ар╕гр╕▓р╕Цр╕╢р╕Зр╕Хр╣Йр╕нр╕Зр╕вр╣Йр╕▓р╕вр╕бр╕▓р╣Ар╕Вр╕╡р╕вр╕Щ ST? р╣Гр╕Щр╣Ар╕бр╕╖р╣Ир╕н Ladder р╕Бр╣Зр╕Фр╕╣р╕Зр╣Ир╕▓р╕вр╕Фр╕╡? р╕Др╕│р╕Хр╕нр╕Ър╕Др╕╖р╕н **"Complexity" (р╕Др╕зр╕▓р╕бр╕Лр╕▒р╕Ър╕Лр╣Йр╕нр╕Щ)** р╕Др╕гр╕▒р╕Ъ

* **Ladder Diagram (LD):** р╣Ар╕Ыр╕гр╕╡р╕вр╕Ър╣Ар╕кр╕бр╕╖р╕нр╕Щр╕Бр╕▓р╕гр╕зр╕▓р╕Ф *Wiring Diagram* р╣Ар╕лр╕бр╕▓р╕░р╕бр╕▓р╕Бр╕кр╕│р╕лр╕гр╕▒р╕Ъ Logic р╣Бр╕Ър╕Ъ Boolean р╕Зр╣Ир╕▓р╕вр╣Ж р╣Ар╕Кр╣Ир╕Щ "р╕Цр╣Йр╕▓р╕Бр╕Фр╕Ыр╕╕р╣Ир╕б A р╣Бр╕ер╕░ Sensor B р╕Чр╕│р╕Зр╕▓р╕Щ р╣Гр╕лр╣Й Motor C р╕лр╕бр╕╕р╕Щ" р╕бр╕▒р╕Щр╣Ар╕лр╣Зр╕Щр╕ар╕▓р╕Ю Power Flow р╕Кр╕▒р╕Фр╣Ар╕Ир╕Щ р╕Кр╣Ир╕▓р╕Зр╣Др╕Яр╣Ар╕Вр╣Йр╕▓р╣Гр╕Ир╕Зр╣Ир╕▓р╕в
* **Structured Text (ST):** р╕Др╕╖р╕н *High-Level Language* р╕Чр╕╡р╣Ир╕бр╕╡р╕гр╕▓р╕Бр╕Рр╕▓р╕Щр╕бр╕▓р╕Ир╕▓р╕Бр╕ар╕▓р╕йр╕▓ Pascal р╣Бр╕ер╕░ C р╕бр╕▒р╕Щр╣Ар╕Бр╕┤р╕Фр╕бр╕▓р╣Ар╕Юр╕╖р╣Ир╕нр╕Ир╕▒р╕Фр╕Бр╕▓р╕гр╕Бр╕▒р╕Ър╕кр╕┤р╣Ир╕Зр╕Чр╕╡р╣И Ladder р╕Чр╕│р╣Др╕Фр╣Йр╕вр╕▓р╕Бр╕лр╕гр╕╖р╕нр╕Чр╕│р╣Др╕бр╣Ир╣Др╕Фр╣Йр╣Ар╕ер╕в р╣Ар╕Кр╣Ир╕Щ:
    * **Complex Math:** р╕Бр╕▓р╕гр╕Др╕│р╕Щр╕зр╕Ур╕кр╕╣р╕Хр╕гр╕Др╕Ур╕┤р╕Хр╕ир╕▓р╕кр╕Хр╕гр╣Мр╕Лр╕▒р╕Ър╕Лр╣Йр╕нр╕Щ р╕Цр╣Йр╕▓р╣Ар╕Вр╕╡р╕вр╕Щр╕Фр╣Йр╕зр╕в Ladder р╕Ър╕ер╣Зр╕нр╕Бр╕Ир╕░р╕Хр╣Ир╕нр╕Бр╕▒р╕Щр╕вр╕▓р╕зр╣Ар╕лр╕вр╕╡р╕вр╕Фр╕Ир╕Щр╕Фр╕╣р╣Др╕бр╣Ир╕гр╕╣р╣Йр╣Ар╕гр╕╖р╣Ир╕нр╕З р╣Бр╕Хр╣Ир╣Гр╕Щ ST р╣Ар╕Вр╕╡р╕вр╕Щр╣Бр╕Др╣Ир╕Ър╕гр╕гр╕Чр╕▒р╕Фр╣Ар╕Фр╕╡р╕вр╕зр╕Ир╕Ъ
    * **Loops & Iterations:** р╕Бр╕▓р╕гр╕зр╕Щр╕ер╕╣р╕Ы `FOR`, `WHILE` р╣Ар╕Юр╕╖р╣Ир╕нр╕Ир╕▒р╕Фр╕Бр╕▓р╕гр╕Вр╣Йр╕нр╕бр╕╣р╕ер╣Гр╕Щ Array р╕Ир╕│р╕Щр╕зр╕Щр╕бр╕▓р╕Б р╕Чр╕│р╣Др╕Фр╣Йр╕Зр╣Ир╕▓р╕вр╣Бр╕ер╕░р╕Бр╕гр╕░р╕Кр╕▒р╕Ър╕Бр╕зр╣Ир╕▓р╕бр╕▓р╕Б
    * **String Manipulation:** р╕Бр╕▓р╕гр╕Хр╕▒р╕Фр╕Хр╣Ир╕нр╕Вр╣Йр╕нр╕Др╕зр╕▓р╕б р╕лр╕гр╕╖р╕нр╕Др╕╕р╕вр╕Бр╕▒р╕Ъ Barcode Scanner/Printer р╕ар╕▓р╕йр╕▓ ST р╕Ир╕▒р╕Фр╕Бр╕▓р╕гр╣Др╕Фр╣Йр╣Ар╕лр╕бр╕╖р╕нр╕Щ C# р╕Др╕гр╕▒р╕Ъ

> **Architect Verdict:** р╣Гр╕Кр╣Й Ladder р╕кр╕│р╕лр╕гр╕▒р╕Ъ Safety Interlock р╕Зр╣Ир╕▓р╕вр╣Ж р╣Бр╕Хр╣Ир╕Цр╣Йр╕▓р╣Ар╕бр╕╖р╣Ир╕нр╣Др╕лр╕гр╣Ир╣Ар╕гр╕┤р╣Ир╕бр╕бр╕╡р╕Бр╕▓р╕гр╕Др╕│р╕Щр╕зр╕У (Calculation), р╕Бр╕▓р╕гр╕зр╕Щр╕ер╕╣р╕Ы (Loops), р╕лр╕гр╕╖р╕нр╕Бр╕▓р╕гр╕Ир╕▒р╕Фр╕Бр╕▓р╕гр╕Вр╣Йр╕нр╕бр╕╣р╕е (Data Handling) р╣Гр╕лр╣Йр╣Ар╕Ыр╕ер╕╡р╣Ир╕вр╕Щр╕бр╕▓р╣Гр╕Кр╣Й ST р╕Чр╕▒р╕Щр╕Чр╕╡р╕Др╕гр╕▒р╕Ъ

## 2. р╕Бр╕Ор╣Ар╕лр╕ер╣Зр╕Бр╕Вр╕нр╕З Syntax (The Grammar of ST)

### 2.1 р╕Бр╕▓р╕гр╕Ир╕Ър╕Ыр╕гр╕░р╣Вр╕вр╕Д (Semicolon)
р╣Ар╕Кр╣Ир╕Щр╣Ар╕Фр╕╡р╕вр╕зр╕Бр╕▒р╕Ъ C# р╕Чр╕╕р╕Бр╕Др╕│р╕кр╕▒р╣Ир╕З (Statement) р╣Гр╕Щ ST р╕Хр╣Йр╕нр╕Зр╕Ир╕Ър╕Фр╣Йр╕зр╕в **Semicolon (;)** р╣Ар╕кр╕бр╕н

```pascal
iCount := iCount + 1; // р╕нр╕вр╣Ир╕▓р╕ер╕╖р╕б ; р╣Ар╕Фр╣Зр╕Фр╕Вр╕▓р╕Ф

```

### 2.2 р╕Бр╕▓р╕гр╕Бр╕│р╕лр╕Щр╕Фр╕Др╣Ир╕▓ (Assignment Operator :=)

р╕Щр╕╡р╣Ир╕Др╕╖р╕нр╕Ир╕╕р╕Фр╕Чр╕╡р╣Ир╕Др╕Щр╣Ар╕Вр╕╡р╕вр╕Щ C# р╕Юр╕ер╕▓р╕Фр╕Ър╣Ир╕нр╕вр╕Чр╕╡р╣Ир╕кр╕╕р╕Ф!

* **C#:** р╣Гр╕Кр╣Й `=` р╣Ар╕Юр╕╖р╣Ир╕н Assign р╕Др╣Ир╕▓
* **ST:** р╣Гр╕Кр╣Й `:=` (Colon-Equal) р╣Ар╕Юр╕╖р╣Ир╕н Assign р╕Др╣Ир╕▓р╕Др╕гр╕▒р╕Ъ

р╕лр╕ер╕▒р╕Бр╕Бр╕▓р╕гр╕Др╕╖р╕н: `р╕Хр╕▒р╕зр╣Бр╕Ыр╕гр╕Чр╕▓р╕Зр╕Лр╣Йр╕▓р╕в := р╕Др╣Ир╕▓р╕Чр╕▓р╕Зр╕Вр╕зр╕▓;` (р╣Ар╕нр╕▓р╕Др╣Ир╕▓р╕Вр╕зр╕▓ р╣Др╕Ыр╕вр╕▒р╕Фр╣Гр╕кр╣Ир╕Хр╕▒р╕зр╣Бр╕Ыр╕гр╕Лр╣Йр╕▓р╕в)

> **тЪая╕П р╕Вр╣Йр╕нр╕Др╕зр╕гр╕гр╕░р╕зр╕▒р╕З:** р╣Ар╕Др╕гр╕╖р╣Ир╕нр╕Зр╕лр╕бр╕▓р╕в `=` р╣Гр╕Щ ST р╣Гр╕Кр╣Йр╕кр╕│р╕лр╕гр╕▒р╕Ър╕Бр╕▓р╕г **р╣Ар╕Ыр╕гр╕╡р╕вр╕Ър╣Ар╕Чр╕╡р╕вр╕Ър╕Др╣Ир╕▓ (Comparison)** р╣Ар╕Чр╣Ир╕▓р╕Щр╕▒р╣Йр╕Щ (р╣Ар╕лр╕бр╕╖р╕нр╕Щ `==` р╣Гр╕Щ C#)

**р╕Хр╕▒р╕зр╕нр╕вр╣Ир╕▓р╕Зр╣Ар╕Ыр╕гр╕╡р╕вр╕Ър╣Ар╕Чр╕╡р╕вр╕Ъ:**

```pascal
// р╕Бр╕▓р╕гр╕Бр╕│р╕лр╕Щр╕Фр╕Др╣Ир╕▓ (Assignment)
MotorSpeed := 1500;  // тЬЕ р╕Цр╕╣р╕Бр╕Хр╣Йр╕нр╕З
MotorSpeed = 1500;   // тЭМ р╕Ьр╕┤р╕Ф (р╕Щр╕╡р╣Ир╕Др╕╖р╕нр╕Бр╕▓р╕гр╣Ар╕Кр╣Зр╕Др╕зр╣Ир╕▓ MotorSpeed р╣Ар╕Чр╣Ир╕▓р╕Бр╕▒р╕Ъ 1500 р╕лр╕гр╕╖р╕нр╣Др╕бр╣И)

// р╕Бр╕▓р╕гр╣Ар╕Ыр╕гр╕╡р╕вр╕Ър╣Ар╕Чр╕╡р╕вр╕Ъ (Comparison in IF)
IF MotorSpeed = 1500 THEN ... // тЬЕ р╕Цр╕╣р╕Бр╕Хр╣Йр╕нр╕З (р╣Ар╕Чр╕╡р╕вр╕Ър╣Ар╕Чр╣Ир╕▓ if (MotorSpeed == 1500))

```

## 4. Hello World: The "Direct Online" Logic
